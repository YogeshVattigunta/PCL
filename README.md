# 🚀 ASEAS – Automated Script Evaluation and Analysis System

## 📌 Overview
ASEAS (Automated Script Evaluation and Analysis System) is an AI-powered platform designed to automate the evaluation of handwritten and typed exam scripts. It reduces manual effort, ensures consistency in grading, and provides detailed feedback to students.

The system focuses on semantic understanding of answers rather than exact keyword matching, making evaluation more intelligent and fair.

---

## 🎯 Objectives
- Automate evaluation of exam papers  
- Reduce human bias in grading  
- Provide detailed performance insights  
- Help students identify weak areas  
- Support learning through feedback  

---

## ⚙️ Features

### 📝 Script Upload
- Upload handwritten or typed answer sheets  
- Supports image and PDF formats  

### 🔍 Text Extraction
- Extracts text using OCR models  
- Handles imperfect handwriting and noise  

### 🧠 AI-Based Evaluation
- Uses NLP models (e.g., BERT) for semantic understanding  
- Compares student answers with multiple reference answers  

### 📊 Smart Scoring
- Assigns marks based on similarity scores  
- Ensures consistent and fair grading  

### 📈 Performance Analysis
- Topic-wise evaluation  
- Identifies strengths and weaknesses  
- Generates insights for improvement  

### 🤖 Feedback System
- Explains why marks were deducted  
- Suggests ways to improve answers  

---

## 🏗️ System Architecture

1. Input Layer  
   - Upload answer scripts  

2. Preprocessing  
   - Image cleaning  
   - Text extraction using OCR  

3. AI Processing  
   - Semantic encoding (BERT)  
   - Answer comparison  

4. Evaluation Engine  
   - Similarity scoring  
   - Marks allocation  

5. Output Layer  
   - Score generation  
   - Feedback and analysis  

---

## 🛠️ Tech Stack

- Python  
- Flask (Backend)  
- NLP Models (BERT / Transformers)  
- OCR Tools  
- HTML, CSS, JavaScript (Frontend)  

---

## 🚀 How It Works

1. User uploads answer sheet  
2. System extracts text from the document  
3. AI model processes and understands answers  
4. Answers are compared with reference solutions  
5. Marks and feedback are generated  
6. Results are displayed to the user
